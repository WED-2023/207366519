    • אודותיי

    קוראים לי רוברט דה נירו. נולדתי וגדלתי בניו יורק – עיר קשוחה, מהירה, אמיתית. מקום שמלמד אותך ישר מההתחלה שאתה צריך לדעת לעמוד על שלך. מאז שאני זוכר את עצמי, רציתי להיות שחקן. לא בגלל הזוהר או הכסף – פשוט כי יש בזה משהו אמיתי .היכולת להיכנס לדמות אחרת, לספר סיפור, לגעת באנשים – זה מה שמשך אותי.

    תחביבים ותחומי עניין

    מעבר למשחק, אני אוהב להבין איך הדברים עובדים – מאחורי המצלמה, על הסט, בתוך הסיפור. אני אוהב קולנוע טוב, אוכל שמכינים באהבה, ואומנות שמצליחה לעצור אותך לרגע. אני גם אוהב מוזיקה – כזו שמרגישים אותה בלב. ובינינו, קצת איגרוף אף פעם לא מזיק. זה לא רק ספורט – זו דרך לחשוב, להרגיש, להיות נוכח.

    דמות מעוררת השראה

    במהלך השנים פגשתי הרבה אנשים, אבל מרלון ברנדו תמיד נשאר השראה בשבילי. הוא היה שם לא רק בתור שחקן, אלא כאדם שהבין את מהות המשחק. הוא לימד אותי שהמשחק האמיתי נמצא בפרטים הקטנים – בהבעה, בשתיקה, במבט. לא צריך להרים את הקול כדי שיקשיבו – צריך להיות אמיתי.

    פלייליסט

    שאלה 1:

    שלושה הבדלים מרכזיים בין גרסאות הweb השונות הינן:

    א. אופן השימוש בתוכן ומגבלותיו:
    Web 1.0 – רשת סטטית, זמינה לקריאה בלבד (Read-Only). הגולשים יכלו רק לצפות בתוכן, ללא כל אפשרות תגובה, שינוי או השפעה עליו.
    Web 2.0 – רשת דינמית, מאפשרת קריאה וכתיבה (Read & Write). המשתמשים יכולים ליצור תוכן, לשתף, להגיב ולהיות חלק פעיל מהאינטרנט.
    Web 3.0 – רשת מבוזרת, מאפשרת גם הרצת פעולות חכמות (Read, Write & Execute). כוללת אינטראקציה עמוקה יותר עם טכנולוגיות כמו בינה מלאכותית ובלוקצ'יין, לצד חוויית משתמש מותאמת אישית.

    ב. טכנולוגיות:
    Web 1.0 – התבסהה על שימוש בסיסי בhtml, ללא שפות כמו JS ולכן, האתרים ברשת היו דפי אינטרנט סטטיים.
    Web 2.0 – לרשת זו התווסף שימוש בJS ו AJAX ובכך, איפשר לרשת להפוך להיות דינאמית.
    Web 3.0 – רשת המשתמשת בכלי AI שונים וטכנולוגיות בלוקצ׳יין.

    ג. שליטה ובעלות על הנתונים:
    Web 1.0 – התוכן נשלט על ידי בעלי האתרים, ללא אפשרות למשתמשים להשפיע על הנתונים.
    Web 2.0 – הפלטפורמות הגדולות (כגון גוגל, פייסבוק וכדומה..) מחזיקות בנתוני המשתמשים ומנהלות את התוכן.
    Web 3.0 – הנתונים מבוזרים באמצעות טכנולוגיות כמו בלוקצ'יין, כך שהמשתמשים שולטים במידע שלהם.


    שאלה 2:

    נבחר בתרחיש שימוש הבא: משתמש מעוניין למצוא באינטרנט מתכון להכנת עוגה מסוימת.
    כעת נסקור את ההבדלים מהסעיף הקודם לכל רשת לתרחיש שימוש זה:

    א. אופן השימוש בתוכן ומגבלותיו:
    Web 1.0 – המשתמש נכנס לאתר ישן שמכיל דף טקסט פשוט עם מתכון לעוגה. אין תגובות ודירוגים, אין תמונות, אין סרטונים – רק תוכן סטטי שאי אפשר לשנות או להשפיע עליו.
    Web 2.0 – המשתמש מחפש "מתכון לעוגה" בגוגל, ומקבל תוצאות מפורומים, בלוגים ואתרי מתכונים עם דירוגים, תגובות, תמונות וסרטונים. הוא יכול גם לשתף מתכון משלו או לשאול שאלה.
    Web 3.0 – עוזר בישול חכם (למשל מבוסס בינה מלאכותית) מציע למשתמש מתכון מותאם אישית לפי ההעדפות שלו (טבעוני, ללא גלוטן וכו'), על סמך רכיבים זמינים בביתו.

    ב. טכנולוגיות:
    Web 1.0 – המתכון מוצג בעזרת דף html בסיסי בלבד.
    Web 2.0 – המתכונים מוצגים בדף מעוצב הרבה יותר בעזרת css והתוכן באתר שאינו סטטי, מאפשר יכולת להשאיר תגובה.
    Web 3.0 – שילוב הAI בחיפוש המתכון, יספק לנו תוצאות מדויקות יותר בהתאם להעדפותינו.

    ג. שליטה ובעלות על הנתונים:
    Web 1.0 – כל המידע שייך לבעלי האתר. אין אינטראקציה או איסוף מידע מהמשתמש
    Web 2.0 – הפלטפורמות אוספות מידע על המשתמש (לדוגמה: מה הוא חיפש, לאילו אתרים נכנס) ומשתמשות בו לפרסום מותאם.
    Web 3.0 – המשתמש שומר על שליטה מלאה בנתונים שלו. הוא יכול לבחור אם לשתף את תחומי העניין שלו כדי לקבל מתכונים מותאמים – מבלי שהמידע ייאגר או יועבר לגורמים מסחריים.
